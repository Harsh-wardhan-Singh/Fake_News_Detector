About This Project

In todayâ€™s world, misinformation spreads faster than ever due to online platforms and social media. This can cause panic, confusion, and false beliefs among people. This project is an AI-powered Fake News Detection System that uses Machine Learning and Natural Language Processing (NLP) techniques to analyze news text. The input is cleaned and preprocessed before being classified using a trained model. The model was trained on a large dataset containing thousands of real and fake news samples, allowing it to recognize patterns commonly found in misinformation. Although the system is not 100% perfect and should not be trusted blindly, it has achieved strong performance results: Accuracy: 93.8% Recall: 0.92 F1 Score: 0.94 The prediction system uses a TF-IDF Vectorizer along with a LinearSVC model for classification. Earlier versions of the project used a Passive Aggressive Classifier, but it was replaced due to lower accuracy and performance. The website provides a clean and simple interface where users can enter a headline or news text into the search bar and instantly receive: A Prediction Label (REAL / FAKE) A Confidence Score A Reliability Category This project aims to promote awareness, encourage fact-checking, and help reduce the spread of misinformation using modern AI technology.
